# ユーザーマニュアル

## はじめに

本マニュアルは、Microsoft 365 経費申請アプリの使用方法を説明します。

## アプリへのアクセス

### Power Apps から
1. [make.powerapps.com](https://make.powerapps.com) にアクセス
2. 「アプリ」から「経費申請アプリ」を選択

### Teams から
1. Teams を開く
2. 経費管理チームの「経費申請」チャネルを選択
3. タブから「経費申請アプリ」を選択

## 画面説明

### ホーム画面

アプリを開くと最初に表示される画面です。

**表示内容:**
- 今月の申請件数
- 承認待ち件数
- 今月の承認済み合計金額
- 最近の申請一覧

**操作:**
- 「新規申請」ボタン → 申請フォームへ
- 申請をタップ → 詳細画面へ

### 経費申請画面

新しい経費を申請する画面です。

**入力項目:**

| 項目 | 説明 | 必須 |
|------|------|------|
| 支出日 | 経費が発生した日付 | ○ |
| 部署 | 所属部署を選択 | ○ |
| 勘定科目 | 経費の種類を選択 | ○ |
| 内容 | 経費の詳細を記入 | ○ |
| 金額（税込） | 税込金額を入力 | ○ |
| 税区分 | 適用される税区分を選択 | ○ |
| インボイス番号 | T+13桁の番号（該当する場合） | |
| 領収書 | 画像またはPDFを添付 | |

**税区分の選び方:**

| 税区分 | 使用場面 |
|--------|----------|
| 10%・適格（インボイス） | インボイス登録事業者からの購入（標準税率） |
| 8%軽減・適格（インボイス） | 食品等の軽減税率適用品目 |
| 10%・経過措置80% | 2023/10〜2026/9の非インボイス取引 |
| 10%・経過措置50% | 2026/10〜2029/9の非インボイス取引 |
| 非課税 | 保険料、住居家賃など |
| 不課税 | 給与、慶弔費など |

**操作:**
- 「下書き保存」→ 入力内容を保存（申請はされません）
- 「申請する」→ 承認者へ申請を送信

### 申請一覧画面

過去の申請を確認・検索できます。

**フィルター機能:**
- キーワード検索（内容で検索）
- 状態フィルター（申請中、承認済みなど）
- 勘定科目フィルター
- 期間フィルター

**ステータスの意味:**

| ステータス | 説明 |
|-----------|------|
| 下書き | 保存のみ、未申請 |
| 申請中 | 承認者の確認待ち |
| 承認済み | 承認完了 |
| 差戻し | 修正が必要（再申請可能） |
| 却下 | 申請が却下された |

### 申請詳細画面

申請の詳細情報を表示します。

**表示内容:**
- 基本情報（日付、部署、科目、内容）
- 金額情報（税込、税抜、消費税額）
- 承認情報（承認者、日時、コメント）
- 添付した領収書

**操作（下書き・差戻し時のみ）:**
- 「編集」→ 内容を修正
- 「削除」→ 申請を削除
- 「再申請」→ 差戻しから再度申請

### 承認管理画面（承認者のみ）

部下からの申請を承認・却下できます。

**承認待ちタブ:**
- 担当部署の承認待ち申請一覧
- 「承認」ボタンで承認
- 「却下」ボタンで却下（理由入力が必要）

**承認履歴タブ:**
- 過去に承認/却下した申請の履歴

### レポート画面

経費の集計・分析ができます。

**機能:**
- 期間選択（開始日〜終了日）
- 科目別円グラフ
- 月別推移グラフ
- 科目別集計表
- Excelエクスポート

## 承認フローについて

### 承認の流れ

```
申請者が申請
    ↓
承認者にTeams通知
    ↓
承認者がTeams/Outlookで確認
    ↓
承認 or 却下（コメント付き）
    ↓
申請者にTeams通知
    ↓
SharePointのステータス更新
```

### 金額による承認ルート

- **5万円未満**: 第一承認者のみ
- **5万円以上**: 第一承認者 → 第二承認者（順次承認）

※ 閾値は部署ごとに設定されています

### 承認通知の確認方法

1. **Teams**: チャット/Approvals アプリに通知
2. **Outlook**: 承認依頼メールが届く
3. **Teams チャネル**: 経費申請チャネルに投稿

## よくある質問 (FAQ)

### Q: 申請を取り消したい
A: 「申請中」の申請は取り消しできません。承認者に連絡してください。

### Q: 領収書を後から追加したい
A: 「下書き」または「差戻し」状態であれば編集から追加できます。

### Q: インボイス番号がわからない
A: 領収書に「T」で始まる13桁の番号が記載されています。記載がない場合は「区分記載」または「経過措置」を選択してください。

### Q: 承認に時間がかかっている
A: 承認者に直接確認するか、管理者にお問い合わせください。

### Q: 税額が合っているか不安
A: 税込金額と税区分を入力すると、消費税額と税抜金額は自動計算されます。元アプリと同じ計算ロジックを使用しています。

## 注意事項

- 添付ファイルは10MBまで
- 対応形式: 画像（JPG, PNG等）、PDF
- 一度「承認済み」になった申請は変更できません
- 経費データは毎日自動バックアップされます
