# データファイル

## qualification_master.csv

資格マスタリストにインポートするデータです。

### インポート手順

#### 方法1: グリッドビューでコピペ（推奨）

1. SharePointで「資格マスタ」リストを開く
2. 右上「グリッドビューで編集」をクリック
3. CSVファイルをExcelで開く
4. ヘッダー行を除いたデータ部分を選択してコピー
5. SharePointのグリッドビューに貼り付け
6. 「グリッドビューの終了」

#### 方法2: Excelテーブルとして接続

1. Excelで新規ブック作成
2. 「データ」→「テキスト/CSVから」
3. `qualification_master.csv`を選択
4. 「読み込み」
5. データをテーブル形式で確認
6. SharePointリストの「グリッドビューで編集」に貼り付け

#### 方法3: Power Automateでバルク登録

1. Power Automateで新規フロー作成
2. トリガー: 手動
3. アクション: 「項目の作成」(SharePoint) をApply to eachでループ
4. CSVの各行をパースして登録

### データ項目

| 列名 | 説明 | 例 |
|------|------|-----|
| 分類 | 資格のカテゴリ | 税務, 会計, IT等 |
| 資格名 | 資格の名称 | 日商簿記検定 |
| 詳細 | 級やレベル | 1級, 2級, - |
| 報奨金 | 合格時の報奨金（円） | 50000 |
| 難易度 | S/A/B/C/D | A |

### 注意事項

- 「-」は詳細がない場合に使用
- 報奨金0は受験料相当額を別途支給の意味（FP3級など）
- 新しい資格を追加する場合はこのCSVも更新してください
